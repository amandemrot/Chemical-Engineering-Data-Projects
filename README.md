# Chemical-Engineering-Data-Projects
Chemical Engineering projects focusing on plant energy optimization and AI-based process modeling using Python and Power BI.
This repository contains two data-driven projects focused on modernizing chemical process operations.

## Project 1: Chemical Plant Energy Consumption Analysis
- **Goal:** Optimize plant energy usage and identify inefficiencies.
- **Tools:** Python (Pandas), Power BI.
- **Key Findings:** Created a dashboard to monitor energy intensity and pinpoint operational spikes in steam and electricity consumption.

## Project 2: ML-Based Prediction of Hydrogen Production
- **Goal:** Predict Hydrogen yield in Steam Methane Reforming (SMR).
- **Tools:** Python (Scikit-learn, Matplotlib).
- **Key Findings:** Developed a Random Forest model with high predictive accuracy (RMSE: ~2.96). Analysis revealed that Temperature is the primary driver (98% importance) for Hydrogen yield.
